# Task 2: Sentiment Analysis using Machine Learning and Deep Learning

## Objective
Build sentiment analysis models to classify IMDb movie reviews as Positive or Negative.

## Dataset
IMDb Movie Reviews Dataset (50,000 reviews)

## Models Used
- Logistic Regression
- Support Vector Machine (SVM)
- LSTM

## Technologies
- Python
- TensorFlow
- Scikit-learn
- NLTK
- Google Colab

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix

## Results
The LSTM model achieved the highest performance compared to the traditional machine learning models.
